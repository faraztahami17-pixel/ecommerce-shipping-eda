# 🏭 Leather Manufacturing & Export Analytics

An end-to-end **Exploratory Data Analysis (EDA)** project analyzing operations of a synthetic leather manufacturing and export company based in **Tamil Nadu, India**.

This project demonstrates the complete data analytics workflow—from data cleaning and exploration to visualization and business insights—using Python and industry-standard libraries.

---

## 📌 Project Overview

The objective of this project is to analyze manufacturing, quality, inventory, shipment, finance, and employee data to identify trends, monitor operational performance, and support business decision-making.

The analysis is performed using realistic synthetic datasets representing a leather manufacturing company operating across multiple production units.
